# Nick Tran Cinematic Portfolio V2 Hi

This version adds:
- full-screen landing page with no text
- scroll-driven section flow
- focused 3D object background per section
- intro / about / skills / projects / resume / contact pages
- profile image support via public/profile.png
- resume support via public/resume.pdf

## Important
Put these files in the public folder:
- `profile.png`
- `resume.pdf`

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
